ChileApellidos

This project analyzes Chilean surnames and displays data regarding distribution, number of people, and other information. It was built using Django-Python for the web framework, with Bootstrap for the frontend. The database used is Postgres, and the server is a Virtual Machine (Linux) hosted on Digital Ocean. The domain was obtained from iwantmyname. For data analysis, the ChatGPT API was used, and data scraping was done with Selenium.
